# Alkatraz Studio website

The source code for this website:
https://alkatrazstudio.net

To build it from the source or run it locally you need Podman installed.
To run it without Podman, explore [yarn.sh](yarn.sh) and [Dockerfile](Dockerfile).


## Run locally

```sh
./yarn.sh local
```

## Build

```sh
./yarn.sh dist
```

The resulting build will be located in the `.output/public` folder.
